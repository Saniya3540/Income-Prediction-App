# Income Prediction App

Machine Learning web application that predicts whether a person's income is above or below 50K using classification algorithms.

## Live Demo
https://income-prediction-app-n2nihzatkjthmkwwtbdatf.streamlit.app

## Features
- Data preprocessing
- Missing value handling
- Outlier treatment
- One Hot Encoding
- Feature Scaling
- Model training and evaluation
- Streamlit deployment

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Machine Learning Algorithms
- KNN Classifier
- Decision Tree
- Random Forest

## Model Performance
- KNN Accuracy: 81%
- Random Forest Accuracy: 81%
- Decision Tree Accuracy: 78%
